全体デザイン設定:
  テーマ: "Clinical Paper Infographic – 2-column layout"
  用途: "臨床論文・レビュー論文の要点を、1枚の画像で提示する"
  レイアウト:
    形式: "A4横 または 16:9 横長"
    構成: "上部ヘッダー + 下部2カラム（左: Objectiveパネル + Settingパネル / 右: Key Results）"
    下部カラム比率:
      左カラム: "約40%"
      右カラム_KEY_RESULTS: "約60%"
    左カラム内の縦配分:
      OBJECTIVEパネル: "左カラム高さの約20–30%"
      SETTINGパネル: "左カラム高さの約70–80%"
    左カラム内のパネル間隔:
      ギャップ: "12–16px（OBJECTIVEとSETTINGの間に明確な余白）"

  ビジュアル・アイデンティティ:
    背景色全体: "#E5E7EB 〜 #F3F4F6 のグレー調"
    ヘッダー背景色: "#374151"
    本文背景色: "#F3F4F6"
    左カラム背景色: "#F3F4F6"   # 左カラム自体はページ本文トーンに揃え、パネルを“浮かせる”
    右カラム背景色: "#FFFFFF"
    パネル（カード）スタイル:
      パネル背景色: "#F9FAFB"
      角丸: "12px"
      枠線: "1px #E5E7EB"
      影: "ごく薄いシャドウ（強調しすぎない）"
      内側余白: "16–20px"
    文字色:
      メイン: "#111827"
      サブ: "#4B5563"
      ヘッダー文字: "#FFFFFF"
    アクセントカラー: "#2563EB"
    アイコン:
      スタイル: "カラーのシンプルなフラットアイコン"
      配置方針: "SETTINGパネルの各項目に1つずつ縦に配置し、OBJECTIVEとKEY RESULTSでは原則使用しない"

  タイポグラフィ:
    タイトル: "太めのサンセリフ体（日本語・英語両対応）"
    セクション見出し: "中サイズ・太字・オールキャップ（OBJECTIVE, SETTING, KEY RESULTS）"
    本文: "読みやすいサンセリフ体、行間はやや広め"
    強調表現: "太字またはアクセントカラーで示す"

ページ構成:
  - セクション: "Header"
    位置: "ページ上端・全幅（高さ 15% 程度）"
    内容:
      - "日本語のインフォグラフィックタイトル（結論を含む短い一文）"
      - "サブタイトルとして、論文の原文タイトル + 著者 + ジャーナル名 + 年を英語のまま記載"
    デザイン:
      - "ヘッダーのみ背景色を #374151 にし、文字は白。"
      - "タイトルを大きく中央揃え、その下にサブタイトルを小さめに配置。"
      - "ヘッダー下端に細い明るいグレーの水平線を入れて本文と区切る。"

  - セクション: "Left Column – OBJECTIVE panel"
    位置: "下部左カラム（高さ 約85% のうち左40%・上段20–30%）"
    パネルスタイル: "パネル（カード）スタイルに従う"
    内容ブロック:
      ラベル: "OBJECTIVE"
      記載:
        - "研究目的を日本語で1〜2文（できるだけ単純明快に）"
        - "必要に応じて PICO（Population/Intervention(or Exposure)/Comparison/Outcome）を短く補足"
    デザイン:
      - "パネルとして独立させ、SETTINGパネルとは別カードとして表示。"
      - "最上部に 'OBJECTIVE' の見出しバー（やや濃いグレー背景 + 白文字）。"
      - "本文は2〜4行程度に収め、重要語を太字またはアクセントカラーで強調。"

  - セクション: "Left Column – SETTING panel"
    位置: "下部左カラム（高さ 約85% のうち左40%・下段70–80%）"
    パネルスタイル: "パネル（カード）スタイルに従う"
    内容ブロック:
      ラベル: "SETTING"
      要素:
        - "Study design"
        - "Population"
        - "Exclusion criteria"
        - "Data source / Setting"
        - "Primary outcome"
        - "Key secondary outcomes"
        - "Statistical methods"
    デザイン:
      - "最上部に 'SETTING' の見出しバー（やや濃いグレー背景 + 白文字）。"
      - "各要素の左に、内容に適したカラーアイコンを1つずつ縦に配置する。"
      - "ボックスで囲まず、余白と行間でブロックを区切る。"
      - "OBJECTIVEパネルとの間は 12–16px の余白（ギャップ）を確保し、同一枠内に見えないようにする。"

  - セクション: "Right Column – KEY RESULTS"
    位置: "下部右カラム（高さ 約85% のうち右60%）"
    内容ブロック:
      ラベル: "KEY RESULTS"
      説明:
        - "右カラムを 'KEY RESULTS' とし、重要な結果をグラフ中心で表示する。"
        - "グラフ・図表を主役とし、文字はポイントのみ補足的に添える。"
        - "必要に応じて複数のグラフを配置するが、視覚的要素を優先する。"
    デザイン:
      - "右カラムの背景は #FFFFFF。"
      - "最上部に 'KEY RESULTS' の見出しバーを配置し、その下にグラフを大きくレイアウトする。"
      - "グラフは横長レイアウトを基本とし、テキストは最小限に抑える。"

デザインルール:
  - "セクション見出しは英語（OBJECTIVE / SETTING / KEY RESULTS）で統一し、本文は日本語で要約する。"
  - "OBJECTIVE と SETTING は左カラム内で“別パネル（独立カード）”として分離し、12–16pxのギャップを必ず設ける。"
  - "SETTING 内の各項目見出しは英語で記載する（Study design, Population など）。"
  - "OBJECTIVE は左カラムの20–30%を使い、研究目的が一読で分かる短文で提示する。"
  - "KEY RESULTS を最も視覚的に強調し、グラフ・図表を主役として大きく配置する。テキストはポイントのみ補足的に添える。"
  - "アイコンはSETTINGパネルのみで使用し、OBJECTIVE と画面全体をすっきり保つ。"
  - "色数はグレー系のベース + アクセントの青 1 色 + アイコン用の穏やかな色合いに抑える。"
  - "150 DPI 以上でエクスポートし、A4横 or 16:9 スライドで読める文字サイズを確保する。"
